# Boult India Admin Panel

Professional admin panel for managing Boult India e-commerce operations.

## Features

- **Order Management**: View, update, and manage customer orders
- **Product Management**: Add, edit, and manage product catalog
- **Invoice Generation**: Professional PDF invoices with company branding
- **Dashboard Analytics**: Real-time order and sales insights
- **Authentication**: Secure admin login system

## Tech Stack

- React 18 with TypeScript
- Tailwind CSS
- React Router DOM
- PDF Generation (jsPDF, html2canvas)
- Lucide React Icons

## Installation

```bash
npm install
npm start
```

## Environment Variables

```env
REACT_APP_API_URL=https://boult-india-bakend-new.onrender.com
```

## Usage

1. Login with admin credentials
2. Manage orders and products
3. Generate professional invoices
4. Monitor business analytics
